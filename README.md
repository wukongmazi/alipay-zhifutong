#### 介绍
支付宝直付通所解决的问题和微信电商收付通类似，不仅是电商行业场景的一把利器，还涉及数娱等互联网平台专属打造的集支付、结算、分账等功能为一体的资金解决方案。该方案致力于帮助平台通过合规的资金清算模式完成平台上各商户的交易资金处理。

![Image text](https://img-blog.csdnimg.cn/1beddeb0db01413091b8f6e3dedddc74.png)

平台通过直付通提供的接口将商户进件为支付宝的二级商户，不仅支持个体工商户、企业等主体类型进件，还支持小微商户即无营业执照的个人商家进件。

具体应用场景如：
1、平台将商户信息进件到支付宝，成为平台的二级商户；
2、商户使用签约支付宝账号登录支付宝，完成支付宝收单结算协议签署；
3、买家下单购买商品之后，支付宝记录二级商户的待结算资金；
4、二级商户发货，买家确认收货后，平台发起确认结算请求，交易资金结算到商户的结算账户中；
5、平台通过分账能力，从商户结算账户中抽取一定佣金到平台账户。

支付宝直付通管理系统已接入进件、分账关系维护、合单支付、资金结算、分账和退款接口，其中二级商户进件功能能可视化操作。
